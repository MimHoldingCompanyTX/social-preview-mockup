# Website Quick Wins Checklist (v1)

Goal: 45–60 min pass to lift perceived quality and conversion.

- [ ] Add canonical OpenGraph/Twitter cards (title, desc, sharing image 1200x630)
- [ ] Add favicon/apple-touch-icon (maskable), theme-color
- [ ] Form: add autocomplete attributes and aria-labels; client-side validation hints
- [ ] Lazy-load large images; ensure width/height set to avoid layout shift
- [ ] Compress hero images (85% JPG or AVIF/WebP); ensure <200KB hero target
- [ ] Add “Booked through” proof badges (optional) or testimonials stub
- [ ] Add bilingual toggle copy to meta description (ES available)
- [ ] Add local business schema.org (Organization + Service)
- [ ] Ensure footer shows business email + Dallas, TX for local trust
- [ ] Add /press kit (bio, headshot, logo ZIP) placeholder
